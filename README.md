# Freelancer Directory API

A RESTful API for managing freelancers, built as a technical assessment for **Etiqa Insurance & Takaful (.NET Intern)**.

## Technologies
- .NET 10
- ASP.NET Core Web API
- Dapper ORM 
- SQL Server LocalDB
- Clean Architecture (Domain, Application, Infrastructure, API)
- Swagger / OpenAPI
